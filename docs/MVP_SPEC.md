# 0. Core Values
1. Easily save a recipe from social media
2. Make the user want to use their saved recipes

## 1. Ingestion
- Accept share extension
- Accept manual video upload
- Accept instagram link
- Save source

## 2. Extraction 
- Dish name (normalized)
- Ingredients (structured)
- Instructions (ordered)
- Missing recipe info protection
- Explicit messages when information is missing

## 3. View Recipe
- Clear details (title, source, description)
- Ingredients list
- Step by step instructions
- Clean UI

## 4. Cook Mode
- Preparation phase
- Focused on one step at a time
- Timers

## 5. Editable recipe screen
- Edit name
- Edit Ingredients
- Edit steps

## 6. Recipe Library
- List saved recipes
- Search by name, ingredient
